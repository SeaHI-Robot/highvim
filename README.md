# HighVim

> My Vim & NeoVim Configuration (developed on linux)
>
> **NeoVim** is more Recommended. Vim shows bugs when doing some keybind configs, which is annoying. 
>
> This configuration contains a **local version** vim & nvim for local machine (nodejs needed), and a simple vimrc **without any plugins**
>
> `vim` branch stores *local* & *no-plugin* version of vim.  `nvim` stores *local* version of nvim.

**main branch**
- Run `deploy_vim_local.sh` to deploy highvim, then do `:Plug Install`.
- Run `deploy_vim_no_plugin.sh` to deploy simple vim configuration for no nodejs environment.

**nvim branch**
- Run `deploy_nvim_local.sh` to deploy highnvim.
- Run `deploy_nvim_no_plugin.sh` to deploy simple nvim configuration for no nodejs environment.
