# HighVim

> My Vim & NeoVim Configuration (for linux only)
> This configuration contains a **local version** for local machine, a **server version** for remote machine, and a simple vimrc for those **machine without nodejs**

- Run `./deploy_<local/server>_highvim.sh` to deploy vim configuration, then do `:Plug Install`.
- Run `./deploy_<local/server>_highneovim.sh` to deploy neovim configuration.
- Run `./deploy_no_nodejs_highvim.sh` to deploy simpele vim configuration for no nodejs environment.
