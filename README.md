# HighVim

> My Vim & NeoVim Configuration (for linux only)
> This configuration contains a local version for local machine, and a server version for remote machine

- Run `./deploy_<local/server>_highvim.sh` to deploy vim configuration, then do `:Plug Install`.
- Run `./deploy_<local/server>_highneovim.sh` to deploy neovim configuration.
